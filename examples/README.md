# Examples

A small gallery of frames the agent could write. They are **static** — pick any one and open it in a browser to get a sense of the aesthetic without invoking the skill.

| File | Mode |
|---|---|
| `01-quiet.html` | A still field — almost nothing, a pulse. The kind of frame written when waiting is the activity. |
| `02-loud.html`  | A single word at 42vmin in Impact yellow. Used when something matters and subtlety would lie. |
| `03-canvas.html` | A generative particle drift. Keeps moving while the agent thinks. |

These are illustrative, not templates. The skill explicitly asks the agent to **avoid templated forms** — every real expression should be a fresh latent-space pull. Read these as ranges of intensity, not as starting points to copy.
